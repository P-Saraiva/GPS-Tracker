O código utilizado no projeto é uma remodelação e atualização de códigos utilizados para propósitos semelhantes, exemplos dos desenvolvedores das bibliotecas utilizadas e autoria própria. Navegando pelo github, encontram-se diversos códigos disponíveis que têm as funcionalidades necessárias para desenvolver o protótipo. Garimpando-os e adaptando-os para se encaixarem nos requisitos, o código final foi criado para preencher as funcionalidades básicas do dispositivo, passível de expansão e mais desenvolvimento. A referência que mais influenciou se encontra no seguinte repositório do GitHub: (https://github.com/mkconer/ESP8266-AdafruitIO). Onde algumas funcionalidades foram aproveitadas para conexão entre o microcontrolador ESP8266 e o broker Adafruit.io.
